## O que é Computação em Nuvem?

Computação em nuvem é a entrega de recursos computacionais (servidores, armazenamento, bancos de dados, redes) sob demanda pela internet, com pagamento conforme o uso.

## Modelos de Serviço

### IaaS (Infrastructure as a Service)

Infraestrutura básica: servidores virtuais, armazenamento e redes. Você gerencia o sistema operacional e aplicações.

**Exemplos:** AWS EC2, Google Compute Engine, Azure VMs

### PaaS (Platform as a Service)

Plataforma gerenciada para deploy de aplicações. Você só se preocupa com o código.

**Exemplos:** Heroku, Vercel, Google App Engine

### SaaS (Software as a Service)

Software pronto para uso, acessado via navegador.

**Exemplos:** Gmail, Google Drive, Slack

## Principais Provedores

| Provedor | Regiões | Diferenciador |
|----------|---------|---------------|
| AWS | 33+ regiões | Maior variedade de serviços |
| Azure | 60+ regiões | Integração com Microsoft |
| Google Cloud | 40+ regiões | Big Data e ML |

## Vantagens

- **Elasticidade:** Escala automaticamente conforme a demanda
- **Pagamento por uso:** Pague apenas pelo que consumir
- **Alta disponibilidade:** Infraestrutura redundante global
- **Foco no negócio:** Sem gastar tempo gerenciando hardware

## Conclusão

A nuvem transformou a forma como construímos e entregamos software. Dominar seus conceitos é essencial para qualquer profissional de tecnologia hoje.
