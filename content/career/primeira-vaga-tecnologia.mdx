## O Cenário

Conseguir a primeira vaga em tecnologia é o maior desafio da carreira. O mercado pede experiência, mas como ter experiência se ninguém te contrata? A chave é estratégia — não é sobre saber tudo, é sobre mostrar potencial.

## O Que Estudar

### Para Frontend

1. **HTML + CSS:** Flexbox, Grid, responsividade
2. **JavaScript (ES6+):** Async/await, promises, manipulação de DOM
3. **Framework:** React (maior demanda) ou Next.js
4. **Controle de versão:** Git + GitHub
5. **Bônus:** TypeScript, testes unitários

### Para Backend

1. **Linguagem:** Java com Spring Boot ou Node.js com Express
2. **API REST:** Verbos HTTP, status codes, JSON
3. **Banco de dados:** SQL (PostgreSQL) + NoSQL (MongoDB)
4. **Autenticação:** JWT, OAuth básico
5. **Bônus:** Docker, testes de API

### Para Dados

1. **SQL:** Joins, subqueries, window functions
2. **Python:** Pandas, NumPy
3. **Visualização:** Matplotlib, Seaborn, Power BI
4. **Estatística:** Média, mediana, desvio padrão, correlação
5. **Bônus:** Machine Learning básico (Scikit-learn), Airflow

## Projetos Que Chamam Atenção

Não faça o mesmo projeto do curso. Crie algo autêntico.

| Projeto | O Que Ensina | Diferencial |
|---------|-------------|-------------|
| API de controle financeiro | CRUD, autenticação, banco | Adicionar categorização por ML |
| Clima tempo com previsão | Consumo de API externa | Adicionar histórico com gráficos |
| Sistema de encurtador de URL | Cache, redirect, analytics | Testar com 1000 requisições |
| Bot para Telegram/Discord | Webhooks, filas, deploy | Colocar em produção no Railway |

## Onde Buscar Vagas

- **LinkedIn:** Ative o #OpenToWork, preencha perfil 100%
- **Programa de estágio:** Nubank, PicPay, iFood, AWS, Google
- **Plataformas:** Gupy, 99Jobs, GeekHunter
- **Comunidades:** Discord de tecnologia, grupos no Telegram
- **Indicação:** A forma mais eficaz — conecte-se com pessoas de empresas alvo

## Como Se Preparar para a Entrevista

```mermaid
flowchart LR
    A[Triagem RH] --> B[Teste Técnico]
    B --> C[Entrevista Técnica]
    C --> D[Behavioral]
    D --> E[Offer]
```

| Etapa | Duração | Como se preparar |
|-------|---------|------------------|
| Triagem RH | 30min | Saiba explicar sua stack e motivação |
| Teste técnico | 2-7 dias | Código limpo, README, testes |
| Entrevista técnica | 1h | Pequeno sistema ao vivo (CRUD simples) |
| Behavioral | 30min | Tenha 3 histórias prontas (desafio, erro, acerto) |

## Dicas Finais

- **Seu currículo tem 6 segundos de atenção** — seja direto, uma página, formato PDF
- **GitHub organizado** — README nos projetos, código limpo, commits descritivos
- **Soft skills importam** — humildade para aprender, comunicação clara, proatividade
- **Não espere estar pronto** — candidate-se mesmo com 60% dos requisitos
- **Rejeição não é pessoal** — cada "não" é aprendizado para o próximo processo

## Conclusão

A primeira vaga é uma questão de estratégia + persistência. Monte um portfólio com projetos autênticos, prepare-se para as etapas do processo e não desista no primeiro não. O mercado é grande o suficiente para quem se prepara com intencionalidade.
